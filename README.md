# Configurations

Personal system and application configuration repository, centered around a modular NixOS setup.

This repo contains:

- a flake-based NixOS configuration
- modular system, hardware, and application modules
- Home Manager-managed user configuration
- external dotfiles for editors and desktop tools
- legacy non-Nix configuration directories
- a few custom/packaged extras

At the moment, the main active target is a **Razer Blade 15 (2023)** running **NixOS**.

## Repository Layout

### `sys/nix/`
Main NixOS flake-based configuration.

Important parts:

- `flake.nix`  
  Entry point for the NixOS configuration and flake inputs.

- `models/`  
  Machine-specific definitions.  
  The active model currently exposed by the flake is:

  - `nixRazer-15`

- `modules/`  
  Reusable modules grouped by purpose:
  - `app/`
  - `env/`
  - `hw/`
  - `ovr/`

This is where most of the system logic lives.

### `app/`
Application-specific configuration files kept outside the Nix modules themselves.

- `app/nvim/`
- `app/emacs/`

These are referenced from Home Manager using out-of-store symlinks.

### `pkgs/`
Custom or bundled package sources and related files.

### `sys/stdLinux/`
Older configuration files.

## Included Components

The current NixOS setup includes modules for things like:

### Hardware
- NVIDIA
- Razer device support
- Bluetooth
- Network configuration
- x86 laptop-specific settings

### Environment
- Nix configuration
- security and PAM/Howdy integration
- Nushell
- GUI base configuration
- Hyprland
- virt-manager / virtualization

### Applications
- development tools such as Git, Direnv, and Haskell tooling
- editors such as Neovim and Emacs
- LaTeX-related tools
- gaming tools such as Steam and Prism Launcher
- internet and communication tools such as Discord, Zoom, OneDrive, Zen Browser, and LocalSend
- media tools such as mpv
- utility apps such as Kitty, Rofi, Thunar, Yazi, OBS Studio, Ollama, and more
